<h2>Установник заблокований!</h2>
<p>Для включення майстра установки гри повинен бути створений файл «ENABLE_INSTALL_TOOL».</p>
<ul>
    <li>В папці includes/ створюємо файл з назвою «ENABLE_INSTALL_TOOL». Ім'я файлу має бути у верхньому регістрі. Сам файл повинен бути без розширення і може бути порожнім.</li>
</ul>
<p>В цілях безпеки настійно рекомендується перейменувати файл або видалити його, коли установка буде завершена. Для надійності можна видалити всю папку «install».</p>
